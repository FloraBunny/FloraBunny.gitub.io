# 空间定向能力自评对空间想想想能力的预测
本研究采用单因素被试间设计，通过网络问卷的形式考察被试对于空间定向能力的自评情况，以此为依据区分自评能力高和自评能力低两个组别。再对每个被试进行心理旋转测验以考察其空间想象能力，期望得出高自评能力的个体的实际空间想象能力也会更强的结论。
